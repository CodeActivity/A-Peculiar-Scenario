A Peculiar Scenario | A Python Adventure Game

A Python Adventure Game in spontaneous development.

Download this project as a .zip file Download this project as a tar.gz file

This is most certainly a Peculiar Scenario....

You've just woken up with no recollection of who you are.

All you have is a map, a note, and a town road before you; and it seems to be deserted.

Coded in delicious Python!

The game is 100% Python, created with the help of the amazing Sublime Text (3).

Text-Based Console!

Want to re-live those MS-DOS memories with a pure black, white text Command Prompt interface? No? Oh - well I guess you should think of it as something else then. Preferably more appealing.

Who's making it?

JakesCode and angrydonkey123 are the current moderators of the repository, both coding the game whenever they have spare time.

Please report any issues. It'll help debugging a HUGE amount.
